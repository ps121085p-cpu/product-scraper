# Product Web Scraper (Python)

## 📌 Overview

This project is a Python web scraper that collects product data from an online shop and exports it into structured CSV/Excel files.

It can be used for:

* Data collection from e-commerce websites
* Price monitoring
* Market analysis
* Building datasets for further processing

---

## ⚙️ Tech Stack

* Python
* requests
* BeautifulSoup
* CSV / Excel export

---

## 🚀 Features

* Scrapes multiple pages automatically
* Extracts product name, price, and link
* Saves data into CSV/Excel file
* Simple and reusable script

---

## 📊 Output

Example fields:

* Product Name
* Price
* Product URL

---

## ▶️ How to run

```bash
pip install -r requirements.txt
python scraper.py
```
